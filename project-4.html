Key-19: Check-in application

Innovative technology project

Key-19 is an innovative check-in application. It features tap-in functionality that utilises NFC technology similar to Myki. This feature allows the user to bring their phone near a check-in point to and the app will record the check-in. Key-19 allows users to check-in groups, which would further increase the efficiency especially when frequently visiting the same location with the same people. Key-19 also has an auto-check in function that uses a wider-ranged technology.
